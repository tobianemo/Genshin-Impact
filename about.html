About

Genshin Impact is a free-to-play action roleplaying game that was published by miHoYo., Ltd., Cognosphere Pte., and d/b/a Hoyoverse and was released on September 28th, 2020.

This game features a fantasy open-word environment and action based combat system using elemental magic, character switching, and gacha monetization system for players to obtain characters, weapons, and other helpful resources. The game can only be played with internet connection and features a mulitplayer mode up to a limited amount of four players in a singular world.
